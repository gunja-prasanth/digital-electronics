# Practice Problems — Up/Down Counters

## Basic

Design up counter.

Answer:
Counts in increasing binary sequence.

---

## Medium

Design down counter.

Answer:
Counts in decreasing binary sequence.

---

## Hard

Design up/down counter.

Answer:

* Use control signal (mode)
* Select logic based on mode
* Combine using multiplexing logic

---

## Interview Question

What is advantage of up/down counter?

Answer:

* Flexible counting
* Can increase or decrease based on control
