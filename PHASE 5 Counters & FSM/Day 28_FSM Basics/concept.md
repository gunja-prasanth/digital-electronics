# Finite State Machines (FSM)

## Definition

A Finite State Machine (FSM) is a sequential circuit that transitions between a finite number of states based on inputs.

---

## Components

* States
* Inputs
* Outputs
* Transition logic

---

## State Diagram

Graphical representation of FSM:

* Circles → States
* Arrows → Transitions
* Labels → Input/output conditions

---

## State Table

Tabular representation:

| Present State | Input | Next State | Output |
| ------------- | ----- | ---------- | ------ |

---

## Types of FSM

### Moore Machine

* Output depends only on state

### Mealy Machine

* Output depends on state and input

---

## State Reduction

* Removing equivalent states
* Reduces hardware complexity

---

## Important Points

* Used in controllers, protocols, sequence detection
* Requires proper state encoding
