# RTL Design – Basic Logic Gates

## Module Name

`basic_gates`

## Description

This module implements fundamental digital logic gates using Verilog.

The following gates are included:

* AND
* OR
* NOT
* XOR
* NAND
* NOR

These gates form the foundation of all digital circuits and are essential for designing combinational and sequential systems.

---

## Inputs

* `A` : Input bit
* `B` : Input bit

## Outputs

* `AND_out` : AND operation
* `OR_out` : OR operation
* `NOT_A` : Inversion of A
* `XOR_out` : XOR operation
* `NAND_out` : NAND operation
* `NOR_out` : NOR operation

---

## Applications

* combinational circuit design
* arithmetic logic units (ALU)
* multiplexers and decoders
* sequential circuit building blocks
