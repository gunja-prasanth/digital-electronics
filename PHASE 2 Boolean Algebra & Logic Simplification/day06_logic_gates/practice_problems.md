# Practice Problems

## Problem 1 (Basic)

Evaluate:

F = A + B'C

Let:

A = 1
B = 0
C = 1

Step:

B' = 1

F = 1 + (1·1) = 1 + 1 = 1

Answer:

F = 1

---

## Problem 2 (Medium)

Simplify:

F = A + AB

Using Boolean law:

A + AB = A(1 + B) = A

Answer:

F = A

---

## Problem 3 (Hard)

Implement XOR using only NAND gates.

Expression:

XOR = (A NAND (A NAND B)) NAND (B NAND (A NAND B))

This is a standard interview question.
