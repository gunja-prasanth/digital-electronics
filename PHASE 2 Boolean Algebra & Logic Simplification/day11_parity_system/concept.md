# Parity Generator and Checker System

This project implements a complete parity-based error detection system.

It consists of:

* Even parity generator
* Odd parity checker

---

# Even Parity Generator

Generates a parity bit such that the **total number of 1s is even**.

---

# Odd Parity Checker

Checks whether received data satisfies **odd parity**.

If parity condition fails → error detected.

---

# Importance

Parity systems are widely used in:

* data communication
* memory systems
* error detection circuits

This project demonstrates both generation and validation of parity.
