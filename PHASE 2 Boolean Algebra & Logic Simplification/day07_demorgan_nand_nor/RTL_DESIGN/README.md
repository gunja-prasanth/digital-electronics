# RTL Design – NAND Universal Logic

## Module Name

`nand_universal`

## Description

This module demonstrates how basic logic functions can be implemented using only NAND gates.

It verifies the concept that NAND is a universal gate.

---

## Implemented Functions

* NOT using NAND
* AND using NAND
* OR using NAND

---

## Inputs

* `A`
* `B`

## Outputs

* `NOT_A`
* `AND_out`
* `OR_out`

---

## Key Concept

Using DeMorgan’s theorem, all logic functions can be constructed using NAND gates.

This principle is widely used in digital hardware design and VLSI implementations.
