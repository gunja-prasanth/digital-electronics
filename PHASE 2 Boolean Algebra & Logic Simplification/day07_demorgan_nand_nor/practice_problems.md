# Practice Problems

## Problem 1 (Basic)

Apply DeMorgan:

(A + B)'

Solution:

(A + B)' = A' · B'

---

## Problem 2 (Medium)

Implement F = A + B using only NAND gates.

Step:

A' = A NAND A
B' = B NAND B

F = A + B = (A' NAND B')

---

## Problem 3 (Hard)

Implement F = A · B using only NOR gates.

Step:

A' = A NOR A
B' = B NOR B

F = A · B = (A' NOR B')
