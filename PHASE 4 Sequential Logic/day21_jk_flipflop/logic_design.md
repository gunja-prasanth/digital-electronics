# Logic Design — JK Flip-Flop

## Characteristic Equation

Q(next) = J·Q' + K'·Q

---

## Explanation

* J·Q' → Sets output
* K'·Q → Holds/reset based on K

---

## JK → D Conversion

We know for D FF:
Q(next) = D

So equate:

D = J·Q' + K'·Q

---

## Key Insight

* JK FF is more flexible than SR
* Can act as:

  * Set FF
  * Reset FF
  * Toggle FF
