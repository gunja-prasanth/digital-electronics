# Logic Design

## Operation Selection

| ALU_Sel | Operation |
| ------- | --------- |
| 00      | ADD       |
| 01      | SUB       |
| 10      | AND       |
| 11      | OR        |

---

# Arithmetic Operations

ADD:

Result = A + B

SUB:

Result = A - B

(Implemented using 2’s complement)

---

# Logic Operations

AND:

Result = A & B

OR:

Result = A | B

---

# System Structure

ALU consists of:

* Adder block
* Logic unit
* Control selection (MUX-like behavior)
