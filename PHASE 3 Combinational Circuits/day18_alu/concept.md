# Arithmetic Logic Unit (ALU)

An ALU is a combinational circuit that performs arithmetic and logical operations.

---

# Operations Supported

* ADD
* SUB
* AND
* OR

---

# Inputs

* A [3:0]
* B [3:0]
* ALU_Sel [1:0]

---

# Output

* Result [3:0]

---

# Importance

ALU is a core component of:

* processors
* microcontrollers
* digital systems

It combines arithmetic + logic operations in one unit.
