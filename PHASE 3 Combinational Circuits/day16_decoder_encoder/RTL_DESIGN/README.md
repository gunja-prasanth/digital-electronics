# RTL Design – Decoder & Encoder

## Modules

* decoder_3to8
* priority_encoder_4to2

---

## Decoder

Converts binary input into one-hot output.

Used in address decoding and memory selection.

---

## Encoder

Converts multiple inputs into binary code.

Priority encoder selects highest priority input.

---

## Key Concept

Decoder outputs represent minterms.

Any Boolean function can be implemented using decoder + OR logic.

---

## Applications

* memory addressing
* interrupt handling
* control logic design
