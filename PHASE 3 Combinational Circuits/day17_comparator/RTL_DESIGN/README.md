# RTL Design – Comparator

## Module Name

`comparator`

## Description

This module compares two binary numbers and determines:

* Equality
* Greater than
* Less than

---

## Inputs

* A [1:0]
* B [1:0]

## Outputs

* EQ
* GT
* LT

---

## Logic

EQ = (A == B)
GT = (A > B)
LT = (A < B)

---

## Applications

* ALU decision logic
* branching conditions
* digital comparison circuits
