# Comparator

A comparator is a combinational circuit that compares two binary numbers.

---

# Outputs

For inputs A and B:

* Equal → A = B
* Greater → A > B
* Less → A < B

---

# 1-bit Comparator

Compare single bits:

A and B

---

# 2-bit Comparator

Compare multi-bit numbers:

A = A1 A0
B = B1 B0

---

# Applications

* ALU decision making
* conditional branching in processors
* digital control systems
