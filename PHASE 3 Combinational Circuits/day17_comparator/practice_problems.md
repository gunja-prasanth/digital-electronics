# Practice Problems

## Problem 1 (Basic)

Design 1-bit comparator.

Solution:

EQ = A'B' + AB
GT = A B'
LT = A' B

---

## Problem 2 (Medium)

Design 2-bit comparator.

Solution:

Compare MSB first, then LSB.

---

## Problem 3 (Hard)

Design comparator using gates.

Use AND, OR, NOT gates based on Boolean expressions.
