# Full Adder

A Full Adder is a combinational circuit that adds **three input bits**:

* A
* B
* Carry-in (Cin)

Outputs:

* Sum (S)
* Carry-out (Cout)

---

# Operation

It performs:

A + B + Cin

---

# Why Full Adder is Important

Unlike Half Adder, Full Adder can:

* handle carry from previous stage
* be cascaded to form multi-bit adders

---

# Example

A = 1, B = 1, Cin = 1

1 + 1 + 1 = 11

Sum = 1
Carry = 1

---

# Applications

* ripple carry adder
* arithmetic logic unit (ALU)
* processor arithmetic operations
