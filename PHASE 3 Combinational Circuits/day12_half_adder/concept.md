# Half Adder

A Half Adder is a combinational circuit used to add two single-bit binary numbers.

Inputs:

* A
* B

Outputs:

* Sum (S)
* Carry (C)

---

# Operation

It performs binary addition:

A + B

Produces:

* Sum → result bit
* Carry → overflow bit

---

# Key Idea

Sum represents the least significant bit.
Carry represents overflow to next stage.

---

# Example

A = 1, B = 1

Binary addition:

1 + 1 = 10

Sum = 0
Carry = 1

---

# Applications

* basic building block of adders
* used in full adders
* used in arithmetic logic units (ALU)
