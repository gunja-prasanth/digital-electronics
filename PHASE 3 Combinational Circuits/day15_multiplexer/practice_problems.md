# Practice Problems

## Problem 1 (Basic)

Design 2:1 MUX.

Answer:

Y = S'I0 + SI1

---

## Problem 2 (Medium)

Design 4:1 MUX using 2:1 MUX.

Solution:

Use 3 multiplexers:

* 2 in first stage
* 1 in second stage

---

## Problem 3 (Hard)

Implement Boolean function using MUX:

F(A,B,C) = Σ(1,2,6,7)

Approach:

* Use variables as select lines
* Map remaining variable to inputs

(This is a common interview question)
