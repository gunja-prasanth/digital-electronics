# Practice Problems

## Problem 1 (Basic)

Convert 45₁₀ to binary.

Solution:

45 / 2 = 22 remainder 1  
22 / 2 = 11 remainder 0  
11 / 2 = 5 remainder 1  
5 / 2 = 2 remainder 1  
2 / 2 = 1 remainder 0  
1 / 2 = 0 remainder 1  

Binary:

45₁₀ = **101101₂**

---

## Problem 2 (Medium)

Convert 101101₂ to decimal.

Solution:

(1×2⁵) + (0×2⁴) + (1×2³) + (1×2²) + (0×2¹) + (1×2⁰)

= 32 + 0 + 8 + 4 + 0 + 1

= **45₁₀**

## Problem 3 (Hard)

Perform binary subtraction:

11001₂ − 1011₂

Step 1: Align numbers

11001
01011
-----

Step 2: Perform subtraction

11001
-01011
------

Borrow method:

Result:

11001₂ − 1011₂ = **1110₂**

Verification:

11001₂ = 25₁₀  
1011₂ = 11₁₀  

25 − 11 = 14

14₁₀ = **1110₂**
