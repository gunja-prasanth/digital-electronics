# Code Converter System

This project integrates multiple digital code conversion modules into a single system.

Modules included:

* Binary to Gray converter
* BCD to Excess-3 converter
* Parity generator

Such integration reflects how real digital systems combine multiple functional blocks into a unified hardware module.

---

# Objective

To design a system that:

* Converts binary input to Gray code
* Converts BCD input to Excess-3
* Generates parity bit for input data

---

# Importance in Digital Systems

Code converters and parity generators are widely used in:

* communication systems
* processor data paths
* error detection circuits
* embedded systems

This project demonstrates modular RTL design and integration of multiple combinational blocks.
